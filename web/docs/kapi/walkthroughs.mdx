# See it in action

Describe the outcome you want — translate this page, ship the app in Japanese,
keep a glossary consistent — and let kapi do the work. Each clip below is a real
command sequence, run for real and recorded exactly; nothing is staged. Pick the
surface you work in.

For where to start, see the [Kapi overview](/kapi/overview); to follow along
yourself, the [CLI recipes](/kapi/cli) run the same commands in your browser.

## Drive it from Claude

Install [the Agent Skill](/kapi/get-started/use-with-skills) and you describe the
result in plain language — Claude runs the right `kapi` commands, reads the output,
and loops until the work passes kapi's gates.
