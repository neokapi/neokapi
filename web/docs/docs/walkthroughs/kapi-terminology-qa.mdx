# Enforce terminology with kapi termbase + term-check

> Driven by [`walkthroughs/kapi-terminology-qa.scene.yaml`](https://github.com/neokapi/neokapi/blob/main/web/docs/walkthroughs/kapi-terminology-qa.scene.yaml) — one source generates both this interactive embed and the recorded video. Do not edit by hand; change the scene spec and regenerate.

A glossary makes terminology consistent across your translations and across locales. `kapi termbase` exposes lookup/search, and `kapi term-check` flags terminology drift in target files before they ship.

## Try it

The in-browser build pre-seeds a termbase from the project glossary (shown as `glossary.csv` in the files pane), so the termbase commands operate on a real backend. Step through the rail.

<KapiGuidedEmbed id="kapi-terminology-qa" />

Inspect the termbase stats, look up a single term, search across concepts, then pseudo-translate `messages_en.json` and run `kapi term-check` to see violations flagged against the seeded terminology.

## Next

Hook the same termbase into a flow with `kapi run` so every push runs the terminology check automatically.
