# Estimate translation cost with word-count

> Driven by [`walkthroughs/kapi-word-count.scene.yaml`](https://github.com/neokapi/neokapi/blob/main/web/docs/walkthroughs/kapi-word-count.scene.yaml) — one source generates both this interactive embed and the recorded video. Do not edit by hand; change the scene spec and regenerate.

You need to estimate the cost of a translation before kicking off a vendor job. `kapi word-count` reads any supported format and outputs a quick breakdown so you can put a number on the bill.

## Try it

A sample `messages.json` is already in the in-browser project folder. Run the step below.

<KapiGuidedEmbed id="kapi-word-count" />

Point `kapi word-count` at a JSON message catalog and watch it report blocks and source words. The output is the kind of number you paste into a quote.
