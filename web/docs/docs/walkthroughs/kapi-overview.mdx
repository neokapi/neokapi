# 30-second tour of kapi

> Driven by [`walkthroughs/kapi-overview.scene.yaml`](https://github.com/neokapi/neokapi/blob/main/web/docs/walkthroughs/kapi-overview.scene.yaml) — one source generates both this interactive embed and the recorded video. Do not edit by hand; change the scene spec and regenerate.

A 30-second tour of what kapi is and what it can do. Useful as the very first thing someone sees on the kapi docs landing page.

## Try it

`kapi` is compiled to WebAssembly and runs entirely in your browser — no install, no server. Step through the tour from the rail, or press **Run all**.

<KapiGuidedEmbed id="kapi-overview" />

`kapi --help` shows the top-level commands. `kapi formats` enumerates the available format readers. `kapi tools` lists the processing tools registry.
