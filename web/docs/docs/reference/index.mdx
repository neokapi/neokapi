# Reference

Generated, authoritative documentation for everything the framework exposes —
commands, formats, and tools — derived directly from the code and the format
and tool schemas.

## Interactive references

These pages are generated from the live registries and schemas:

- [Command Reference](/commands) — every `kapi` command, with runnable examples.
- [Format Reference](/formats) — every format reader/writer and its configurable
  parameters.
- [Tool Reference](/tools) — every processing tool and its inputs and outputs.
- [Flow Visualization](/flow-visualization) — built-in flows rendered as graphs.

## Project status

- [Parity](/parity) — coverage against the Okapi Framework.
- [Benchmarks](/pseudobench) — performance measurements.
- [Test Results](/test-comparison) — filter test comparison dashboard.

:::note

Per-entry reference pages (one page per command, format, and tool) are
generated under `reference/commands/`, `reference/formats/`, and
`reference/tools/`. Until then, use the interactive references above.

:::
