# Try It in Your Browser

Every command on this page runs the **real `kapi` binary** — compiled to
WebAssembly — directly in your browser. There is no server and nothing leaves
your machine. Press **▸ Run** on any command to open the playground; it boots
once on the first run, then stays warm.

## Pseudo-translate a file

Pseudo-translation replaces source text with accented look-alikes so you can
spot untranslated strings and layout problems before real translation. The
command seeds a small `messages.json` and writes `out.json`:

<RunnableSnippet
  cmd="kapi pseudo-translate messages.json -o out.json"
  seed={["messages.json"]}
/>

After it runs, open `out.json` in the **Files** panel to preview the result, or
download it.

## Count the words

Estimate translation volume for the same file:

<RunnableSnippet cmd="kapi word-count messages.json" seed={["messages.json"]} />

## Explore the formats kapi understands

This one needs no input file — it lists every registered format:

<RunnableSnippet cmd="kapi formats list" />

## Edit, then run yourself

With `editable`, the command is placed at the prompt instead of running
automatically — press **Enter** when you are ready, or tweak it first:

<RunnableSnippet cmd="kapi word-count messages.json --json" seed={["messages.json"]} editable />
