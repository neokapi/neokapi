# Get oriented in kapi

**Goal:** find your way around the CLI — what kapi can read, what it can do to
content, and where to look next. This is the first thing to run when you open a
new terminal.

`kapi` is compiled to WebAssembly and runs entirely in your browser here — no
install, no server. Step through the tour from the rail, or press **Run all**.

<KapiGuidedEmbed id="kapi-overview" />

Three commands orient you:

- `kapi --help` lists the top-level commands.
- `kapi formats` enumerates the format readers kapi can parse and write.
- `kapi tools` lists the processing tools registry — the transforms you can run
  on extracted content.
