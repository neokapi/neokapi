# Guides

Each guide takes a single task — estimate a job, catch UI truncation, enforce a
glossary — states the goal, walks the steps, and lets you run the commands
yourself. The interactive embeds run the real `kapi` binary compiled to
WebAssembly in your browser: no install, no server, no API key. Every command
shown is the command that runs.
