# Walkthroughs

Each walkthrough is a short, recorded run of a real kapi workflow — one command
sequence, start to finish. They're generated from prompts under
[`web/docs/guides/`](https://github.com/neokapi/neokapi/tree/main/web/docs/walkthroughs)
and the terminal recordings are captured in CI, so what you see is exactly what
the CLI does.
