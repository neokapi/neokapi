Creating a Project​

Translation Editor​

The translation editor shows source and target side-by-side with live preview.

Focus View​

For distraction-free editing, use the focus view which shows one segment at a time.

Translation Memory​

Browse and search your translation memory entries.

TM Leverage in the Editor​

Open a project with existing TM entries, then use "TM Lookup" to auto-fill translations. The Context panel shows per-block TM matches with scores and one-click apply.

Terminology Explorer​

Full concept-oriented terminology management. Browse concepts, add terms with lifecycle statuses, import from CSV or JSON, search and filter by locale and domain.

Context Panel: TM + Terminology​

The editor's Context panel shows both TM matches and terminology suggestions for the current block. This is the primary way translators interact with linguistic resources during editing.

End-to-End Translation Workflow​

Full workflow from project creation through translation to export.

Flow Editor​

View and customize translation pipelines.

Settings​

Configure AI providers for machine translation.

Workspace Switcher​

Navigate between workspace views using the sidebar with navigation panel.

The sidebar provides quick access to all workspace views: Translate, Termbase, Memory, Flows, Connectors, and Settings. The workspace name is displayed at the top, and the sidebar can be collapsed for more screen space.

Account and Authentication​

Configure AI providers and service connections for your workspace.

For team collaboration with a bowrain-server, authentication is handled via OIDC/SSO. CLI authentication is available via bowrain auth login — see the CLI demo videos for details.

Workspace Project Management​

Create and manage multiple projects within a workspace.

The project dashboard shows all workspace projects with locale and file information. Create new projects, add files, and quickly switch between projects.
