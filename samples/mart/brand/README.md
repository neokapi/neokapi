# Mart brand voice

The Mart fiction has a plain, practical voice aimed at small-shop owners. It
avoids marketing inflation. `forbidden-terms.txt` lists words a brand check
should flag; the table below gives preferred substitutions for the most common
offenders.

| Forbidden | Use instead |
| --- | --- |
| leverage | use |
| utilize | use |
| game-changing | useful, or describe the concrete benefit |
| synergy | (drop it; name the actual benefit) |

Naming rules:

- The product name (`KapiMart`, `OkapiMart`, or `BowMart` depending on the
  surface) is do-not-translate. Keep it in Latin script in every locale.
- Prefer "AI assistant" over "Copilot", "bot", or "agent" in customer-facing
  copy.

`forbidden-terms.txt` is one term per line, suitable as input to a brand check.
