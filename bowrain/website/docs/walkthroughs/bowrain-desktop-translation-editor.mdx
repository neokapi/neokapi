User launches the desktop app. Workspace rail shows the seeded project. Click into the project, land on the file list, click into `i18n/en.json`. The translation editor opens with three blocks (welcome / save / cancel). User clicks the target cell of "Welcome", types "Bienvenue", clicks save. Toast confirms.
