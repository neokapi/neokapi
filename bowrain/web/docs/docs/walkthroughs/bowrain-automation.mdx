# Automate translation sync with local rules + GitHub Actions

> Generated from [`walkthroughs/bowrain-automation.md`](https://github.com/neokapi/neokapi/blob/main/bowrain/web/docs/walkthroughs/bowrain-automation.md). Do not edit by hand — change the prompt and regenerate.

Bowrain has two layers of automation: **local rules** declared on the `.kapi` recipe (top-level `hooks:` and `automations:`) that run on the developer machine (e.g. pre-push QA), and **GitHub Actions workflows** that run on the CI/CD side. Together they make translation sync fully automated — code lands, translations follow.
