# Invite a teammate to the workspace

> Generated from [`walkthroughs/bowrain-web-invite-workflow.md`](https://github.com/neokapi/neokapi/blob/main/bowrain/web/docs/walkthroughs/bowrain-web-invite-workflow.md). Do not edit by hand — change the prompt and regenerate.

Workspace owner sends an email invite to a teammate. The invite arrives via email; the recipient clicks the link, signs in, and is added to the workspace.
