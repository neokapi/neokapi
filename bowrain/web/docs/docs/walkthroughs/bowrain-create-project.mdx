# Pseudo-translate an HTML page

> Generated from [`walkthroughs/bowrain-create-project.md`](https://github.com/neokapi/neokapi/blob/main/bowrain/web/docs/walkthroughs/bowrain-create-project.md). Do not edit by hand — change the prompt and regenerate.

A small example showing how to take an existing HTML page, run a pseudo-translation pass with `kapi`, and inspect the result. Useful as a "can I see what the platform does" exercise before you commit to setting up a full Bowrain project.
