# Bootstrap a bowrain project with init

> Generated from [`walkthroughs/bowrain-init.md`](https://github.com/neokapi/neokapi/blob/main/bowrain/web/docs/walkthroughs/bowrain-init.md). Do not edit by hand — change the prompt and regenerate.

`kapi init` scaffolds a `.kapi` project in your repo (the bowrain equivalent of `git init`): a `<dir-name>.kapi` recipe at the project root with a `server:` block, plus a sibling `.kapi/` state directory that holds the block store, sync cache, and TM/termbase.
