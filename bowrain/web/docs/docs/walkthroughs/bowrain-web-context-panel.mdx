# Context panel — TM and terminology while you translate

> Generated from [`walkthroughs/bowrain-web-context-panel.md`](https://github.com/neokapi/neokapi/blob/main/bowrain/web/docs/walkthroughs/bowrain-web-context-panel.md). Do not edit by hand — change the prompt and regenerate.

The editor's context panel surfaces TM matches and terminology hits inline next to the active block, so the translator never has to leave the row to check whether the project has already translated a similar string or whether a glossary term applies.

The walkthrough opens an item in the editor (it lands in the Visual view by default), switches to the Table view, then toggles the **Context** button in the toolbar. The right-hand panel slides in with sections for **TM Matches** and **Terminology**, populated by the engine for whatever block is currently selected.
