# Getting Started

Welcome to the **Acme** documentation. This guide will help you get up and running quickly.

## What is Acme?

Acme is a developer platform that provides the tools and infrastructure you need to build, deploy, and monitor applications at any scale.

## Quick Start

1. **Install Acme** using your preferred package manager
2. **Create a project** with the CLI or web dashboard
3. **Deploy** your first application in under 5 minutes

## Prerequisites

Before you begin, make sure you have:

- Node.js 18 or later
- A free Acme account
- Your favorite code editor
