Check the activity feed for recent changes that need PM attention.

Use `list_projects` and `get_project` to review current status across all
projects. Look for:
- New content pushes (Developer pushed blocks that need task assignment)
- Completed translation batches (translators finished work)
- QA results (quality checks completed, issues found)
- Stalled projects (no activity in 24+ hours)

Also check `email.listInbox` for:
- Translator availability updates or blockers
- QA escalations requiring PM decision
- Developer updates about upstream changes

If there are new content pushes without corresponding translator assignments,
create tasks and notify the team. If nothing needs attention, exit quietly.
