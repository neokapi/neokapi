## Workspace Switcher

Navigate between workspaces using the Slack-inspired sidebar with workspace icon rail and navigation panel.

The workspace UI follows a three-column layout:

| Column | Width | Content |
|--------|-------|---------|
| Workspace Rail | 60px | Workspace icons, create button, user avatar |
| Navigation Sidebar | 220px | View menu (Translate, Termbase, Memory, Flows, Connectors, Settings) |
| Main Content | flex | Active view content |

Key features:
- **Workspace rail** with colored icons for each workspace
- **Switching** between workspaces updates the project list and navigation
- **Creating** a new workspace via the "+" button opens a dialog
- **Account menu** with user info and sign-out

## Account and Authentication

Login flow for connecting Bowrain to a gokapi-server for team collaboration.

The authentication flow works as follows:

1. Open **Settings** and enter your gokapi server URL
2. Click **Sign in with SSO** — your browser opens the server's Dex login page
3. Authenticate with your identity provider (Google, GitHub, LDAP, etc.)
4. The server issues a JWT token and your session is established
5. Server workspaces appear in the workspace rail

For CLI authentication, see the [CLI Auth guide](/docs/user-guide/cli/auth).
