## TM Leverage in the Editor

Open a project with existing TM entries, then use "TM Lookup" to auto-fill translations. The Context panel shows per-block TM matches with scores and one-click apply.

<video controls width="100%" style={{maxWidth: '800px'}}>
  <source src="/video/bowrain/tm-leverage-workflow.webm" type="video/webm" />
</video>

**Walkthrough:**
1. Open `website-translation.kaz` from the sample projects
2. Click on `index.html` to open the translation editor
3. Click "TM Lookup" in the toolbar — blocks with TM matches are auto-filled
4. Click "Context" in the toolbar to open the Context panel
5. Navigate between blocks — the panel shows TM matches with scores
6. Click "Apply" on a match to insert it into the current block

## Terminology Explorer

Full concept-oriented terminology management. Browse concepts, add terms with lifecycle statuses, import from CSV or JSON, search and filter by locale and domain.

<video controls width="100%" style={{maxWidth: '800px'}}>
  <source src="/video/bowrain/term-explorer.webm" type="video/webm" />
</video>

**Walkthrough:**
1. Open `software-ui.kaz` from the sample projects
2. Click "Terminology" in the project view
3. Browse the termbase — each concept shows terms across locales with status badges
4. Use the search bar to filter by term text
5. Click "Add Concept" to create a new concept with terms in multiple locales
6. Click "Import CSV" to import terms from `sample-terms.csv`
7. Edit any concept by clicking on it — modify terms, domain, definition, or status

## Context Panel: TM + Terminology

The editor's Context panel shows both TM matches and terminology suggestions for the current block. This is the primary way translators interact with linguistic resources during editing.

<video controls width="100%" style={{maxWidth: '800px'}}>
  <source src="/video/bowrain/context-panel-workflow.webm" type="video/webm" />
</video>

**Walkthrough:**
1. Open `website-translation.kaz` and go to the translation editor
2. Click "Context" in the toolbar to open the Context panel
3. Navigate to a block containing terminology (e.g., "cloud storage", "encryption")
4. The TM section shows matches with source, target, score, and match type
5. The Terminology section shows matched terms with target suggestions and status
6. Click "Apply" on a TM match to insert the translation
7. Use terminology suggestions to ensure consistent term usage across the project
