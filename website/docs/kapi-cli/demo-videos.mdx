## Terminology Management

Import termbases, look up terms in running text, search concepts, and view statistics.

:::note[Video coming soon]
This demo will be auto-generated once the termbase CLI end-to-end test is added.
:::

```bash
# Import terms from CSV
kapi termbase import terms.csv --format csv -s en -t fr --domain general

# Look up terms in a sentence
kapi termbase lookup "The authentication module uses end-to-end encryption" -s en -t fr

# Search the termbase
kapi termbase search "encryption" -s en --domain security

# View termbase statistics
kapi termbase stats
```
