# CLI Demo Videos

Animated demos of common `kapi` CLI workflows.

## Overview

Quick tour of kapi's capabilities.

<video controls autoPlay loop muted width="100%" style={{maxWidth: '900px'}}>
  <source src="/video/cli/overview.webm" type="video/webm" />
</video>

```bash
kapi --help
kapi formats
kapi tools
```

## Format Conversion

Convert between JSON, YAML, XML, and other formats.

<video controls autoPlay loop muted width="100%" style={{maxWidth: '900px'}}>
  <source src="/video/cli/convert.webm" type="video/webm" />
</video>

```bash
kapi convert messages.json -f json --to yaml
```

## Word Count

Estimate translation costs with word counting.

<video controls autoPlay loop muted width="100%" style={{maxWidth: '900px'}}>
  <source src="/video/cli/word-count.webm" type="video/webm" />
</video>

```bash
kapi word-count messages.json
```

## Pseudo-Translation

Generate pseudo-translations to test UI for truncation and RTL issues.

<video controls autoPlay loop muted width="100%" style={{maxWidth: '900px'}}>
  <source src="/video/cli/pseudo-translate.webm" type="video/webm" />
</video>

```bash
kapi pseudo-translate messages.json --target-lang fr
```

## Create Translation Project

Pack source files into a `.kaz` project for Bowrain.

<video controls autoPlay loop muted width="100%" style={{maxWidth: '900px'}}>
  <source src="/video/cli/create-project.webm" type="video/webm" />
</video>

```bash
kapi pack -i landing-page.html \
  --source-lang en \
  --target-lang fr,de \
  -o website.kaz
```
