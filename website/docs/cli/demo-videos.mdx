## Format Conversion

Convert between JSON, YAML, XML, and other formats.

<video controls width="100%" style={{maxWidth: '900px'}}>
  <source src="/video/cli/convert.webm" type="video/webm" />
</video>

```bash
# Convert JSON to YAML
kapi convert messages.json -f json --to yaml

# Convert YAML to JSON
kapi convert messages.yaml -f yaml --to json
```

## Regenerating Videos

Videos are generated from VHS tape files in `docs/tapes/`:

```bash
# Generate all CLI demos
make cli-recordings

# Or manually
cd docs/tapes && ./generate.sh
```

Requires VHS (`brew install charmbracelet/tap/vhs`) and a local terminal with TTY access.
