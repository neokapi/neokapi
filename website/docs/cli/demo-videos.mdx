# CLI Demo Videos

Quick examples of common `kapi` CLI workflows.

:::tip Generate Demo Videos
Run `make cli-recordings` locally to generate animated demos from the VHS tape files.
:::

## Overview

```bash
# See all available commands
kapi --help

# List supported formats
kapi formats

# List available tools
kapi tools
```

## Format Conversion

Convert between JSON, YAML, XML, and other formats.

```bash
# Convert JSON to YAML
kapi convert messages.json -f json --to yaml

# Convert YAML to JSON  
kapi convert messages.yaml -f yaml --to json

# Pipe conversions
cat data.json | kapi convert -f json --to yaml
```

## Word Count

Estimate translation costs with word counting.

```bash
kapi word-count messages.json
kapi word-count src/*.html --recursive
```

## Pseudo-Translation

Generate pseudo-translations to test UI for truncation and RTL issues.

```bash
kapi pseudo-translate messages.json --target-lang fr
kapi pseudo-translate src/strings.json -o src/strings.fr.json
```

## Create Translation Project

Pack source files into a `.kaz` translation project for Bowrain.

```bash
# Pack files into a project
kapi pack -i landing-page.html -i about.html \
  --source-lang en \
  --target-lang fr,de \
  -o website.kaz

# Unpack after translation
kapi unpack website.kaz -o output/
```

## Generating Video Demos

Video demos are generated using [VHS](https://github.com/charmbracelet/vhs) from tape files in `docs/tapes/`.

```bash
# Install VHS
brew install charmbracelet/tap/vhs

# Generate all demos (requires local terminal)
make cli-recordings
```

VHS requires a local terminal with TTY access - won't work in SSH or CI.
