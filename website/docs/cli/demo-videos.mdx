Import, export, and look up translation memory entries.

```bash
# Import TMX file

kapi tm export --db project.tm -o output.tmx

# Look up a segment in the TM
kapi tm lookup --db project.tm "Welcome to our platform" -s en -t fr
```

## Terminology Management

Import termbases, look up terms in running text, search concepts, and view statistics.

:::note[Video coming soon]
This demo will be auto-generated once the termbase CLI end-to-end test is added.
:::

```bash
# Import terms from CSV
kapi termbase import terms.csv --format csv -s en -t fr --domain general

# Look up terms in a sentence
kapi termbase lookup "The authentication module uses end-to-end encryption" -s en -t fr

# Fuzzy lookup
kapi termbase lookup "authenticating users" -s en -t fr --fuzzy

# Search the termbase
kapi termbase search "encryption" -s en --domain security

# View termbase statistics
kapi termbase stats
```

## Terminology-Aware Translation Flow

Run a pipeline that uses both TM and terminology to enrich translations.

:::note[Video coming soon]
This demo will be auto-generated once the terminology-aware flow end-to-end test is added.
:::

```bash
# Translation flow with terminology lookup
kapi flow run --input docs/ --output out/ \
  --tools term-lookup,tm-leverage,ai-translate \
  -s en -t fr \
  --termbase project.tb

# Post-translation terminology enforcement
kapi flow run --input translated/ --output qa/ \
  --tools term-enforce \
  -s en -t fr \
  --termbase project.tb
```
