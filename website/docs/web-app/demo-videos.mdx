# Demo Videos

Short screencasts demonstrating key features of the gokapi web application.

:::info
Screencasts for the web application are coming soon. The web app shares the same core components (translation editor, TM explorer, terminology explorer) as the [Bowrain desktop app](/docs/bowrain/demo-videos). See the Bowrain demo videos for screencasts of the shared editing experience.
:::

## Shared Components

The web application and Bowrain desktop app use the same `@gokapi/ui` shared component library. These Bowrain screencasts show the same editing experience you get in the web app:

- **Translation editor** — grid mode, focus mode, inline editing, tag handling
- **TM Explorer** — browse, search, add, edit, delete TM entries
- **Terminology Explorer** — browse, search, add, edit, delete concepts
- **Context panel** — TM matches and terminology suggestions per block
- **Progress tracking** — color-coded status bar

See [Bowrain Demo Videos](/docs/bowrain/demo-videos) for these screencasts.

## Web-Specific Features

The following features are unique to the web application and will have dedicated screencasts:

- **OIDC login flow** — authenticating through Dex with GitHub, Google, or LDAP
- **Workspace management** — creating workspaces, switching between them, workspace rail navigation
- **File upload** — drag-and-drop file upload with format auto-detection
- **Multi-user collaboration** — multiple users working in the same workspace
