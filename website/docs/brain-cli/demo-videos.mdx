## Project Workflow

End-to-end project workflow: inspect content, count words, and pseudo-translate.

<video controls autoPlay loop muted width="100%" style={{maxWidth: '900px'}}>
  <source src="/video/brain/create-project.webm" type="video/webm" />
</video>

```bash
# Initialize a project
brain init --name "Landing Page" --source en --targets fr

# Count words for cost estimation
kapi word-count landing-page.html

# Pseudo-translate for UI testing
kapi pseudo-translate landing-page.html --target-lang fr

# Open in web UI for visual editing
brain serve
```

## Workspace Management

Create and manage workspaces via the REST API for team collaboration.

<video controls autoPlay loop muted width="100%" style={{maxWidth: '900px'}}>
  <source src="/video/brain/workspaces.webm" type="video/webm" />
</video>

```bash
# List your workspaces
curl -H "Authorization: Bearer $TOKEN" \
  https://gokapi.example.com/api/v1/workspaces

# Create a team workspace
curl -X POST -H "Authorization: Bearer $TOKEN" \
  -d '{"name": "Acme Translations", "slug": "acme"}' \
  https://gokapi.example.com/api/v1/workspaces

# Create a project in the workspace
curl -X POST -H "Authorization: Bearer $TOKEN" \
  -d '{"name": "Website", "source_locale": "en", "target_locales": ["fr","de"]}' \
  https://gokapi.example.com/api/v1/workspaces/acme/projects
```

## Terminology Management

Import termbases, look up terms in running text, search concepts, and view statistics.

:::note[Video coming soon]
This demo will be auto-generated once the termbase CLI end-to-end test is added.
:::

```bash
# Import terms from CSV
brain termbase import terms.csv --format csv -s en -t fr --domain general

# Look up terms in a sentence
brain termbase lookup "The authentication module uses end-to-end encryption" -s en -t fr

# Fuzzy lookup
brain termbase lookup "authenticating users" -s en -t fr --fuzzy

# Search the termbase
brain termbase search "encryption" -s en --domain security

# View termbase statistics
brain termbase stats
```
